Web Basics

HTML / CSS / JavaScript を触ってみよう

HTMLで要素を作り、CSSで見た目を整え、JavaScriptで画面を更新します。

1. HTML はページの部品を作る

この見出し、文章、入力欄、ボタンはすべてHTMLの要素です。 id は1つの要素を指定したいとき、 class は同じ種類の要素をまとめて指定したいときに使います。

2. CSS は見た目を整える

この枠は .lesson-card と .highlight-box というclassで装飾されています。 ページ上部のheaderは #page-header というidでも指定できます。

  • タグ名指定: body, h1, button
  • class指定: .lesson-card, .primary-button
  • id指定: #message, #update-button
